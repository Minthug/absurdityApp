# absurdityApp
AbsurdityApp - 형제자매를 위한 심부름 플랫폼 👨‍👧‍👦
프로젝트 소개
AbsurdityApp은 형제자매 간의 심부름을 재미있게 만드는 새로운 방식의 플랫폼입니다.
배달 앱처럼 심부름을 등록하고, 적절한 보상과 함께 심부름을 수행하며 형제자매 간의 유대감을 강화하는 것이 목적입니다.
개발 동기

새로운 기술 스택 학습 및 실전 적용
개인 프로젝트를 통한 기술적 성장
형제자매 간 소통 방식의 새로운 제안
